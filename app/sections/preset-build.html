Preset builds

matcha.css provides preset builds for added convenience. If you need a specific set of features, you can also use the custom builder.

  • Default
    • All semantic styling and extra features except @istanbul-coverage.
    • <link rel="stylesheet" href="https://matcha.mizu.sh/matcha.css">
  • Lite
    • All semantic styling, @break-words, and @discrete-scrollbars.
    • <link rel="stylesheet" href="https://matcha.mizu.sh/matcha.lite.css">
  • Utility classes
    • @root and @utilities.
    • <link rel="stylesheet" href="https://matcha.mizu.sh/matcha.utilities.css">
  • Istanbul coverage
    • @root, @syntax-highlighting, and @istanbul-coverage.
    • <link rel="stylesheet" href="https://matcha.mizu.sh/matcha.istanbul.css">
